# /components - Reusable UI components

All shared React components for Marketiv.

Structure:

- /ui: design system primitives (button, input, card, badge)
- /layout: structural wrappers (header, footer, dashboard shell)
- /sections: marketing and landing sections
- /animations: GSAP animation wrappers and helpers
- /marketplace: product listing and discovery components
- /dashboard: UMKM and Creator dashboard components
- /checkout: payment and order flow components

Rules:

- Keep components domain-focused and reusable.
- Do not couple UMKM-only components to Creator-only flows.
